This is my project, I build it in order to learn Node.js, Express.js and Mongoose.
I created this project using the following technologies:

Front-End:

JQuery, Underscore.js, Backbone.js, Marionette, Require.js, Socket.io, Handlebars, LESS and Twitter-Bootstrap

Back-End:

Node.js, Express.js, Socket.io, Mongoose and MongoDB
I built this in 2 and a half weeks. I am sure I can add a lot more to this, if time permits.
Next challenge will be to figure out how this can scale properly. Meaning, "what if we had thousands of users?"
